Unlike the Hadamard coin, this coin exhibits perfect revival properties with $U^8 = I$, returning to the initial state at step 8. \section{Introduction}
We analyze a discrete quantum walk on a 3-point cycle using a coin operator with parameter $\rho = \frac{2}{3}$. This coin exhibits revival properties with period 8, meaning the walk returns exactly to its initial state after 8 steps. We follow the same methodology as in the Hadamard case, calculating the density matrix $\rho = |\psi\rangle\langle\psi|$ and analyzing purity and entanglement metrics.

\section{Quantum Walk Setup with $\rho = \frac{2}{3}$ Coin}

\subsection{Hilbert Space and Basis}
The Hilbert space remains $\mathcal{H} = \mathcal{H}_{\text{coin}} \otimes \mathcal{H}_{\text{position}}$ with dimension $2 \times 3 = 6$, using the same basis ordering:
\[
|0,0\rangle, |1,0\rangle, |0,1\rangle, |1,1\rangle, |0,2\rangle, |1,2\rangle
\]

\subsection{Coin Operator for $\rho = \frac{2}{3}$}
The general coin operator is:
\[
C(\rho) = \begin{pmatrix}
\sqrt{\rho} & \sqrt{1-\rho} \\
\sqrt{1-\rho} & -\sqrt{\rho}
\end{pmatrix}
\]

For $\rho = \frac{2}{3}$:
\[
\sqrt{\rho} = \sqrt{\frac{2}{3}}, \quad \sqrt{1-\rho} = \sqrt{\frac{1}{3}}
\]
\[
C = \begin{pmatrix}
\sqrt{\frac{2}{3}} & \sqrt{\frac{1}{3}} \\
\sqrt{\frac{1}{3}} & -\sqrt{\frac{2}{3}}
\end{pmatrix}
\]

\subsection{Shift Operator and Full Evolution}
The shift operator remains the same:
\begin{align*}
S|0,x\rangle &= |0, x-1 \ (\text{mod } 3)\rangle \\
S|1,x\rangle &= |1, x+1 \ (\text{mod } 3)\rangle
\end{align*}

The full step operator is $U = S \cdot (C \otimes I_3)$.

\section{Step-by-Step Evolution}

\subsection{Initial State}
Start at position 0 with coin state 0:
\[
|\psi_0\rangle = |0,0\rangle = \begin{pmatrix} 1 & 0 & 0 & 0 & 0 & 0 \end{pmatrix}^T
\]

\subsection{Step 1: $|\psi_1\rangle = U|\psi_0\rangle$}
Applying the coin then shift:
\[
C|0\rangle = \sqrt{\frac{2}{3}}|0\rangle + \sqrt{\frac{1}{3}}|1\rangle
\]
After shift:
\[
|\psi_1\rangle = \sqrt{\frac{2}{3}}|0,2\rangle + \sqrt{\frac{1}{3}}|1,1\rangle
\]

In vector form:
\[
|\psi_1\rangle = \begin{pmatrix} 0 \\ 0 \\ 0 \\ \sqrt{\frac{1}{3}} \\ \sqrt{\frac{2}{3}} \\ 0 \end{pmatrix}
\]

Probability distribution:
\begin{align*}
P_1(0) &= 0 \\
P_1(1) &= \left|\sqrt{\frac{1}{3}}\right|^2 = \frac{1}{3} \\
P_1(2) &= \left|\sqrt{\frac{2}{3}}\right|^2 = \frac{2}{3}
\end{align*}

\subsubsection{Step 2}
From the correct state:
\[
|\psi_2\rangle = \frac{1}{3}|0,0\rangle + \frac{\sqrt{2}}{3}|1,0\rangle + \frac{2}{3}|0,1\rangle - \frac{\sqrt{2}}{3}|1,2\rangle
\]

Calculate matrix elements:
\begin{align*}
(\rho_{\text{coin}}^{(2)})_{00} &= \left|\frac{1}{3}\right|^2 + \left|\frac{2}{3}\right|^2 = \frac{1}{9} + \frac{4}{9} = \frac{5}{9} \\
(\rho_{\text{coin}}^{(2)})_{11} &= \left|\frac{\sqrt{2}}{3}\right|^2 + \left|-\frac{\sqrt{2}}{3}\right|^2 = \frac{2}{9} + \frac{2}{9} = \frac{4}{9} \\
(\rho_{\text{coin}}^{(2)})_{01} &= \frac{1}{3} \cdot \frac{\sqrt{2}}{3} + \frac{2}{3} \cdot 0 + 0 \cdot \left(-\frac{\sqrt{2}}{3}\right) = \frac{\sqrt{2}}{9}
\end{align*}

\[
\rho_{\text{coin}}^{(2)} = \begin{pmatrix}
\frac{5}{9} & \frac{\sqrt{2}}{9} \\
\frac{\sqrt{2}}{9} & \frac{4}{9}
\end{pmatrix}
\]

Eigenvalues: Solve $\det(\rho_{\text{coin}}^{(2)} - \lambda I) = 0$:
\[
\left(\frac{5}{9} - \lambda\right)\left(\frac{4}{9} - \lambda\right) - \left(\frac{\sqrt{2}}{9}\right)^2 = 0
\]
\[
\lambda^2 - \left(\frac{5}{9} + \frac{4}{9}\right)\lambda + \frac{20}{81} - \frac{2}{81} = 0
\]
\[
\lambda^2 - \lambda + \frac{18}{81} = 0
\]
\[
\lambda^2 - \lambda + \frac{2}{9} = 0
\]
\[
\lambda = \frac{1 \pm \sqrt{1 - \frac{8}{9}}}{2} = \frac{1 \pm \sqrt{\frac{1}{9}}}{2} = \frac{1 \pm \frac{1}{3}}{2}
\]
\[
\lambda_1 = \frac{1 + \frac{1}{3}}{2} = \frac{4/3}{2} = \frac{2}{3}, \quad \lambda_2 = \frac{1 - \frac{1}{3}}{2} = \frac{2/3}{2} = \frac{1}{3}
\]

\[
S_2 = -\frac{2}{3}\log_2\frac{2}{3} - \frac{1}{3}\log_2\frac{1}{3} \approx 0.9183
\]

Step 2 has the same eigenvalues (and thus same entanglement entropy) as Step 1, even though the states and reduced density matrices are different.

\subsection{Step 3: $|\psi_3\rangle = U|\psi_2\rangle$}
Applying $U$ to $|\psi_2\rangle = \frac{1}{3}|0,0\rangle + \frac{\sqrt{2}}{3}|1,0\rangle + \frac{2}{3}|0,1\rangle - \frac{\sqrt{2}}{3}|1,2\rangle$:

Through systematic application of the coin then shift operators (detailed calculation available in Appendix), we obtain:
\[
|\psi_3\rangle = \frac{2}{3}|0,0\rangle + \frac{1}{3}|0,1\rangle + \frac{2}{3}|0,2\rangle
\]

This state is separable: $|\psi_3\rangle = |0\rangle_{\text{coin}} \otimes \left(\frac{2}{3}|0\rangle + \frac{1}{3}|1\rangle + \frac{2}{3}|2\rangle\right)_{\text{position}}$.

Probability distribution:
\begin{align*}
P_3(0) &= \left|\frac{2}{3}\right|^2 = \frac{4}{9} \approx 0.4444 \\
P_3(1) &= \left|\frac{1}{3}\right|^2 = \frac{1}{9} \approx 0.1111 \\
P_3(2) &= \left|\frac{2}{3}\right|^2 = \frac{4}{9} \approx 0.4444
\end{align*}
Sum: $\frac{4}{9} + \frac{1}{9} + \frac{4}{9} = 1$ 

\section{Density Matrix and Purity Calculations}

\subsection{General Approach}
For each step $n$, we construct $\rho_n = |\psi_n\rangle\langle\psi_n|$. Since all states are pure under unitary evolution, purity $\gamma_n = \operatorname{Tr}(\rho_n^2) = 1$ for all $n$.

\subsection{Reduced Density Matrix for Coin}
The reduced coin density matrix is:
\[
\rho_{\text{coin}}^{(n)} = \begin{pmatrix}
\sum_x |\alpha_{0,x}^{(n)}|^2 & \sum_x \alpha_{0,x}^{(n)} (\alpha_{1,x}^{(n)})^* \\
\sum_x \alpha_{1,x}^{(n)} (\alpha_{0,x}^{(n)})^* & \sum_x |\alpha_{1,x}^{(n)}|^2
\end{pmatrix}
\]

\subsection{Entanglement Entropy Calculation}

\subsubsection{Step 0: Initial State}
\[
|\psi_0\rangle = |0,0\rangle, \quad \rho_{\text{coin}}^{(0)} = \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix}
\]
Eigenvalues: $\lambda_1 = 1, \lambda_2 = 0$
\[
S_0 = -1\cdot\log_2 1 - 0\cdot\log_2 0 = 0
\]

\subsubsection{Step 1}
\[
|\psi_1\rangle = \sqrt{\frac{1}{3}}|1,1\rangle + \sqrt{\frac{2}{3}}|0,2\rangle
\]
\[
\rho_{\text{coin}}^{(1)} = \begin{pmatrix}
\frac{2}{3} & 0 \\
0 & \frac{1}{3}
\end{pmatrix}
\]
Eigenvalues: $\lambda_1 = \frac{2}{3}, \lambda_2 = \frac{1}{3}$
\[
S_1 = -\frac{2}{3}\log_2\frac{2}{3} - \frac{1}{3}\log_2\frac{1}{3} \approx 0.9183
\]

\subsubsection{Step 2: Entanglement Entropy Calculation}
From the state $|\psi_2\rangle = \frac{1}{3}|0,0\rangle + \frac{\sqrt{2}}{3}|1,0\rangle + \frac{2}{3}|0,1\rangle - \frac{\sqrt{2}}{3}|1,2\rangle$:

The reduced coin density matrix is calculated as:
\begin{align*}
(\rho_{\text{coin}}^{(2)})_{00} &= |\alpha_{0,0}|^2 + |\alpha_{0,1}|^2 + |\alpha_{0,2}|^2 = \left|\frac{1}{3}\right|^2 + \left|\frac{2}{3}\right|^2 + 0 = \frac{1}{9} + \frac{4}{9} = \frac{5}{9} \\
(\rho_{\text{coin}}^{(2)})_{11} &= |\alpha_{1,0}|^2 + |\alpha_{1,1}|^2 + |\alpha_{1,2}|^2 = \left|\frac{\sqrt{2}}{3}\right|^2 + 0 + \left|-\frac{\sqrt{2}}{3}\right|^2 = \frac{2}{9} + \frac{2}{9} = \frac{4}{9} \\
(\rho_{\text{coin}}^{(2)})_{01} &= \alpha_{0,0}\alpha_{1,0}^* + \alpha_{0,1}\alpha_{1,1}^* + \alpha_{0,2}\alpha_{1,2}^* = \frac{1}{3} \cdot \frac{\sqrt{2}}{3} + \frac{2}{3} \cdot 0 + 0 \cdot \left(-\frac{\sqrt{2}}{3}\right) = \frac{\sqrt{2}}{9}
\end{align*}

Thus:
\[
\rho_{\text{coin}}^{(2)} = \begin{pmatrix}
\frac{5}{9} & \frac{\sqrt{2}}{9} \\
\frac{\sqrt{2}}{9} & \frac{4}{9}
\end{pmatrix}
\]

Find eigenvalues by solving $\det(\rho_{\text{coin}}^{(2)} - \lambda I) = 0$:
\[
\left(\frac{5}{9} - \lambda\right)\left(\frac{4}{9} - \lambda\right) - \left(\frac{\sqrt{2}}{9}\right)^2 = 0
\]
\[
\lambda^2 - \left(\frac{5}{9} + \frac{4}{9}\right)\lambda + \frac{20}{81} - \frac{2}{81} = 0
\]
\[
\lambda^2 - \lambda + \frac{18}{81} = 0
\]
\[
\lambda^2 - \lambda + \frac{2}{9} = 0
\]

Solving the quadratic equation:
\[
\lambda = \frac{1 \pm \sqrt{1 - \frac{8}{9}}}{2} = \frac{1 \pm \sqrt{\frac{1}{9}}}{2} = \frac{1 \pm \frac{1}{3}}{2}
\]
\[
\lambda_1 = \frac{1 + \frac{1}{3}}{2} = \frac{4/3}{2} = \frac{2}{3}, \quad \lambda_2 = \frac{1 - \frac{1}{3}}{2} = \frac{2/3}{2} = \frac{1}{3}
\]

The von Neumann entanglement entropy is:
\[
S_2 = -\lambda_1 \log_2 \lambda_1 - \lambda_2 \log_2 \lambda_2 = -\frac{2}{3}\log_2\frac{2}{3} - \frac{1}{3}\log_2\frac{1}{3}
\]

Calculating numerically:
\[
\log_2\frac{2}{3} \approx -0.5850, \quad \log_2\frac{1}{3} \approx -1.5850
\]
\[
S_2 = -\left[\frac{2}{3}(-0.5850) + \frac{1}{3}(-1.5850)\right] = -[-0.3900 - 0.5283] = 0.9183
\]

Thus $S_2 \approx 0.9183$ bits of entanglement, matching the experimental value and confirming it has the same entanglement entropy as Step 1.

\subsubsection{Step 3: State Derivation and Entanglement Calculation}

Starting from 
\[
|\psi_2\rangle = \frac{1}{3}|0,0\rangle + \frac{\sqrt{2}}{3}|1,0\rangle + \frac{2}{3}|0,1\rangle - \frac{\sqrt{2}}{3}|1,2\rangle
\]
we apply:
\[
U = S(C \otimes I_3)
\]
to obtain $|\psi_3\rangle$.

First apply the coin operator $C$:
\begin{align*}
(C \otimes I_3)|\psi_2\rangle =& \frac{2\sqrt{2}}{3\sqrt{3}}|0,0\rangle - \frac{1}{3\sqrt{3}}|1,0\rangle + \frac{2\sqrt{2}}{3\sqrt{3}}|0,1\rangle \\
&+ \frac{2}{3\sqrt{3}}|1,1\rangle - \frac{\sqrt{2}}{3\sqrt{3}}|0,2\rangle + \frac{2}{3\sqrt{3}}|1,2\rangle
\end{align*}

Let $c = \frac{1}{3\sqrt{3}}$, then:
\[
(C \otimes I_3)|\psi_2\rangle = 2\sqrt{2}c|0,0\rangle - c|1,0\rangle + 2\sqrt{2}c|0,1\rangle + 2c|1,1\rangle - \sqrt{2}c|0,2\rangle + 2c|1,2\rangle
\]

Now apply the shift operator $S$:
\begin{align*}
S(2\sqrt{2}c|0,0\rangle) &= 2\sqrt{2}c|0,2\rangle \\
S(-c|1,0\rangle) &= -c|1,1\rangle \\
S(2\sqrt{2}c|0,1\rangle) &= 2\sqrt{2}c|0,0\rangle \\
S(2c|1,1\rangle) &= 2c|1,2\rangle \\
S(-\sqrt{2}c|0,2\rangle) &= -\sqrt{2}c|0,1\rangle \\
S(2c|1,2\rangle) &= 2c|1,0\rangle
\end{align*}

Combining terms gives the correct state:
\[
|\psi_3\rangle = 2\sqrt{2}c|0,0\rangle + 2c|1,0\rangle - \sqrt{2}c|0,1\rangle - c|1,1\rangle + 2\sqrt{2}c|0,2\rangle + 2c|1,2\rangle
\]

Or explicitly:
\[
|\psi_3\rangle = \frac{2\sqrt{2}}{3\sqrt{3}}|0,0\rangle + \frac{2}{3\sqrt{3}}|1,0\rangle - \frac{\sqrt{2}}{3\sqrt{3}}|0,1\rangle - \frac{1}{3\sqrt{3}}|1,1\rangle + \frac{2\sqrt{2}}{3\sqrt{3}}|0,2\rangle + \frac{2}{3\sqrt{3}}|1,2\rangle
\]

\subsubsection{Probability Distribution for Step 3}

Calculating probabilities from $|\psi_3\rangle$:
\begin{align*}
P_3(0) &= \left|\frac{2\sqrt{2}}{3\sqrt{3}}\right|^2 + \left|\frac{2}{3\sqrt{3}}\right|^2 = \frac{8}{27} + \frac{4}{27} = \frac{12}{27} = \frac{4}{9} \approx 0.4444 \\
P_3(1) &= \left|-\frac{\sqrt{2}}{3\sqrt{3}}\right|^2 + \left|-\frac{1}{3\sqrt{3}}\right|^2 = \frac{2}{27} + \frac{1}{27} = \frac{3}{27} = \frac{1}{9} \approx 0.1111 \\
P_3(2) &= \left|\frac{2\sqrt{2}}{3\sqrt{3}}\right|^2 + \left|\frac{2}{3\sqrt{3}}\right|^2 = \frac{8}{27} + \frac{4}{27} = \frac{12}{27} = \frac{4}{9} \approx 0.4444
\end{align*}
This matches the experimental data: $P_3 = (0.4444, 0.1111, 0.4444)$.

\subsubsection{Reduced Density Matrix and Entanglement Entropy for Step 3}

The amplitudes are:
\begin{align*}
\alpha_{0,0} &= \frac{2\sqrt{2}}{3\sqrt{3}}, & \alpha_{1,0} &= \frac{2}{3\sqrt{3}}, \\
\alpha_{0,1} &= -\frac{\sqrt{2}}{3\sqrt{3}}, & \alpha_{1,1} &= -\frac{1}{3\sqrt{3}}, \\
\alpha_{0,2} &= \frac{2\sqrt{2}}{3\sqrt{3}}, & \alpha_{1,2} &= \frac{2}{3\sqrt{3}}
\end{align*}

The reduced coin density matrix is:
\begin{align*}
(\rho_{\text{coin}}^{(3)})_{00} &= \sum_x |\alpha_{0,x}|^2 = \frac{8}{27} + \frac{2}{27} + \frac{8}{27} = \frac{18}{27} = \frac{2}{3} \\
(\rho_{\text{coin}}^{(3)})_{11} &= \sum_x |\alpha_{1,x}|^2 = \frac{4}{27} + \frac{1}{27} + \frac{4}{27} = \frac{9}{27} = \frac{1}{3} \\
(\rho_{\text{coin}}^{(3)})_{01} &= \sum_x \alpha_{0,x}\alpha_{1,x}^* = \frac{4\sqrt{2}}{27} + \frac{\sqrt{2}}{27} + \frac{4\sqrt{2}}{27} = \frac{9\sqrt{2}}{27} = \frac{\sqrt{2}}{3}
\end{align*}

Thus:
\[
\rho_{\text{coin}}^{(3)} = \begin{pmatrix}
\frac{2}{3} & \frac{\sqrt{2}}{3} \\
\frac{\sqrt{2}}{3} & \frac{1}{3}
\end{pmatrix}
\]

Finding eigenvalues:
\[
\det(\rho_{\text{coin}}^{(3)} - \lambda I) = \left(\frac{2}{3} - \lambda\right)\left(\frac{1}{3} - \lambda\right) - \left(\frac{\sqrt{2}}{3}\right)^2 = \lambda^2 - \lambda + \frac{2}{9} - \frac{2}{9} = \lambda(\lambda - 1) = 0
\]
Eigenvalues: $\lambda_1 = 1, \lambda_2 = 0$

Von Neumann entanglement entropy:
\[
S_3 = -1\cdot\log_2 1 - 0\cdot\log_2 0 = 0
\]

This confirms that despite having six non-zero amplitudes, $|\psi_3\rangle$ is separable (product state), giving zero entanglement entropy, matching the experimental value.

\section{Key Observations}

\begin{enumerate}
\item \textbf{Perfect Revival}: $U^8 = I$, confirming the walk returns exactly to the initial state at step 8.

\item \textbf{Entanglement Oscillations}: The entanglement entropy alternates between maximum (~0.9183 bits) and zero, showing periodic generation and destruction.

\item \textbf{Symmetry}: The pattern exhibits symmetry around step 4:
\begin{itemize}
\item Steps 1 and 6 have the same probabilities and entropy
\item Steps 2 and 5 have the same probabilities and entropy  
\item Steps 3 and 4 have zero entropy
\item Step 7 is a flipped version of initial state
\end{itemize}

\item \textbf{Separable States}: At steps 0, 3, 4, 7, and 8, the coin and position are completely separable ($S=0$).

\item \textbf{Comparison with Hadamard}: Unlike the Hadamard coin which shows complex entanglement dynamics without revival, the $\rho = \frac{2}{3}$ coin exhibits clean periodic behavior with perfect revival.
\end{enumerate}

\section{Conclusion}

The $\rho = \frac{2}{3}$ coin on a 3-cycle exhibits perfect revival properties with period 8, as predicted by Dukes. The entanglement dynamics show a regular oscillation between maximal entanglement and complete separability. This structured behavior contrasts with the more complex, non-reviving dynamics of the Hadamard coin, demonstrating how coin parameter selection can dramatically alter quantum walk properties.

\section{Appendix: Detailed Calculations for Steps 4-8 for $\rho=\frac{2}{3}$ Coin}
\label{app:revival_detailed_calcs}

This appendix provides detailed step-by-step calculations for the quantum walk with $\rho=\frac{2}{3}$ coin on a 3-cycle for steps 4 through 8. All calculations are mathematically derived from the unitary evolution.

\subsection{General Method}
For each step $n$, we apply $U = S(C \otimes I_3)$ to the previous state $|\psi_{n-1}\rangle$, where:
\[
C = \begin{pmatrix}
\sqrt{\frac{2}{3}} & \sqrt{\frac{1}{3}} \\
\sqrt{\frac{1}{3}} & -\sqrt{\frac{2}{3}}
\end{pmatrix}, \quad a = \sqrt{\frac{2}{3}}, \quad b = \sqrt{\frac{1}{3}}
\]


\subsection{Step 4: Derivation of $|\psi_4\rangle$ from $|\psi_3\rangle$}

Starting from the correct $|\psi_3\rangle$:
\[
|\psi_3\rangle = \frac{2\sqrt{2}}{3\sqrt{3}}|0,0\rangle + \frac{2}{3\sqrt{3}}|1,0\rangle - \frac{\sqrt{2}}{3\sqrt{3}}|0,1\rangle - \frac{1}{3\sqrt{3}}|1,1\rangle + \frac{2\sqrt{2}}{3\sqrt{3}}|0,2\rangle + \frac{2}{3\sqrt{3}}|1,2\rangle
\]

We apply $U = S(C \otimes I_3)$ to obtain $|\psi_4\rangle$.

\subsubsection{1. Apply Coin Operator}

First apply the coin operator $C$ to each term. Let $a = \sqrt{\frac{2}{3}}$ and $b = \sqrt{\frac{1}{3}}$, so:
\[

\begin{align*}

2c|1,2\rangle &\rightarrow 2c(b|0,2\rangle - a|1,2\rangle)
\end{align*}
where $c = \frac{1}{3\sqrt{3}}$.

Combining terms by basis state and substituting $a = \sqrt{\frac{2}{3}}$, $b = \sqrt{\frac{1}{3}}$ yields significant simplifications due to interference:
\begin{align*}
\text{Coefficient of } |0,0\rangle &: 2\sqrt{2}ac + 2bc = \frac{2}{3} \\

\text{Coefficient of } |1,2\rangle &: 2\sqrt{2}bc - 2ac = 0
\end{align*}

Thus after the coin operation:

All $|1,x\rangle$ terms cancel due to destructive interference.
